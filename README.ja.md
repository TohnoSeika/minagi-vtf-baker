# Minagi VTF Baker ✦

> [English](README.md) · [中文](README.zh.md)

画像を **Left 4 Dead 2 のスプレー** に変換——簡単で、きれいで、速い。

PNG / JPG / BMP / WEBP 画像を **L4D2 互換の VTF スプレーファイル** に変換する Windows デスクトップツール。  
静止スプレー、動画スプレー、距離スプレーの 3 モードに対応。  
Tauri ベース——軽量でエレガント。

---

## ✨ 特徴

- 🖼️ **3 つのスプレーモード** — 静止 VTF / 動画スプレー / 距離スプレー
- 🎨 **DXT 圧縮** — DXT1 / DXT5 / RGBA8888、アルファチャンネル対応
- 🔄 **サンプリング方式** — 異方性 / バイリニア / 最近傍
- ✏️ **シャープ化** — シャープネスフィルター（オプション）
- 🌐 **多言語 UI** — 简体中文 · 日本語 · English
- 🎨 **テーマカラー** — 好きな色に変更可能
- 🖼️ **カスタム背景** — 背景画像を設定可能
- 🪟 **ウィンドウ透明度** — UI と背景の透明度を個別に調整
- 🚀 **パフォーマンス** — インストーラ約 6MB、即時起動

---

## 📖 使い方

| 操作 | 説明 |
|:---|:---|
| 🖱️ **画像をドロップ** | 左パネルにドラッグ＆ドロップ、またはクリックしてファイル選択 |
| 🔧 **パラメータ設定** | 圧縮形式、出力サイズ、サンプリング方式などを選択 |
| ✨ **変換** | 「変換開始」をクリック — `.vtf` ファイルが元ディレクトリに保存されます |
| 🎬 **動画スプレー** | 「動画VTF」タブで JPG フレームを追加し、動画スプレーを生成 |
| 🌄 **距離スプレー** | 「距離VTF」タブで距離別にレイヤーを追加し、距離スプレーを生成 |
| ⚙️ **カスタマイズ** | 設定ページでテーマ色、背景画像、透明度を変更 |
